# Project Overview

## Business Context
Olist is a Brazilian e-commerce marketplace that connects customers, sellers, and logistics operations. In a marketplace environment, business success depends not only on sales growth, but also on seller quality, product performance, and delivery reliability.

## Project Objective
The objective of this project is to use SQL to analyze marketplace performance using cleaned gold-layer data. The project is designed to simulate the work of a business/data analyst supporting commercial and operational decision-making.

## Main Analysis Areas
- Marketplace sales performance
- Customer geography
- Seller revenue and operational performance
- Product/category performance
- Delivery reliability
- Root-cause analysis of delivery delays

## Analytical Approach
The project uses:
- order-level analysis for customer and delivery KPIs
- item-level analysis for seller and product/category KPIs
- filtered delivery analysis to avoid misleading interpretations from very small sample sizes
